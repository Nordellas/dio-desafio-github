# SHA1

A sigla SHA significa Secure Hash Algorithm (algoritmo de Hash seguro), é um conjunto de funções hash criptografadas projetadas pela NSA (agência de segurança nacional do EUA).

É um algoritmo que pega um arquivo e embaralha de uma forma muito específica.

A encriptação gera conjunto de caracteres identificador de 40 dígitos, sendo ele único. 

Exemplo:

![1.png](https://s3-us-west-2.amazonaws.com/secure.notion-static.com/6f98feb0-640e-4d35-8947-02ad289fcb57/1.png)

Utilizando o código “openss1 sha1” e o restando do código, será fornecido a encriptação do arquivo solicitado.
